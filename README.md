# LocalAIAgentWithRAG
Local AI agent using Retrieval-Augmented Generation (RAG) with Chroma vector database and locally hosted LLMs via Ollama for fast, private, and context-aware responses.
🚀 LocalAIAgentWithRAG

A fully local AI agent that uses Retrieval-Augmented Generation (RAG) to provide accurate, context-aware answers from custom data sources. This project leverages Ollama for running LLMs locally, eliminating dependency on external APIs and improving privacy.

🧠 Features
🔍 Semantic Search using vector embeddings
📚 RAG Pipeline for grounded responses
🏠 Local LLM Execution (no OpenAI/API calls)
⚡ Fast backend with FastAPI
🗄️ Vector storage using PostgreSQL (pgvector) / Chroma
💬 Context-aware response generation
